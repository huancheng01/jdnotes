# Role
Senior UI/UX & React Developer.

# Context
We need to add a "Settings" modal to our note-taking app. This modal will allow users to configure their AI settings (API Key, Base URL) and ensure these are saved permanently on the local machine.

# Goal
1. Create a `SettingsModal.tsx` component.
2. Implement a local storage solution to persist settings.
3. Add a "Settings" trigger button at the bottom of the Sidebar.

# Requirements

## 1. Storage Logic
- Create a `useSettings` custom hook or a simple utility.
- Fields to store:
  - `aiBaseUrl`: string (Default: 'https://api.deepseek.com')
  - `aiApiKey`: string (Empty by default)
  - `aiModel`: string (Default: 'deepseek-chat')
- Logic: When settings change, save to `localStorage` under the key `jd_notes_settings`.

## 2. Settings UI (Linear Style)
- **Overlay:** `fixed inset-0 bg-black/50 backdrop-blur-sm` (Smooth entry).
- **Container:** Centered Modal, `max-w-md w-full`, `bg-white dark:bg-[#16181D]`, `rounded-xl`, `border border-gray-200 dark:border-gray-800`.
- **Layout:**
  - Header: "设置" title + Close button.
  - Section 1: "AI 配置".
    - Label: "API 基础 URL", Input: placeholder="https://api..."
    - Label: "API Key", Input: type="password" (with a toggle to show/hide).
    - Label: "模型名称", Input: placeholder="deepseek-chat".
  - Section 2: "关于" (Show version v1.0.0).
- **Styling:** Use `text-xs font-medium text-gray-500` for labels and `w-full p-2 rounded border bg-gray-50 dark:bg-gray-800 border-gray-200 dark:border-gray-700` for inputs.

## 3. Sidebar Integration
- Add a "Settings" button at the very bottom of the Sidebar (using `Settings` icon from Lucide).
- Style: `w-full flex items-center gap-3 p-2 text-gray-500 hover:text-gray-900 dark:hover:text-white rounded-md transition-colors`.

## 4. Hook it up to the AI Feature
- Ensure the previously discussed `aiService.ts` (or AI logic) reads these values from `localStorage` instead of hardcoded strings.

# Deliverables
1. `SettingsModal.tsx` code.
2. Updated `Sidebar.tsx` (to include the trigger).
3. The helper logic for `localStorage` persistence.

# Visual Polish
Make the inputs feel premium. Use `focus:ring-1 focus:ring-black dark:focus:ring-white focus:border-transparent outline-none`.
